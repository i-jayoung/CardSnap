# CardSnap - 信用卡桌面助手

截图识别信用卡信息，自动生成精美卡片样式和二维码，支持浮窗钉到桌面。

## 功能特性

- **截图识别** - `Ctrl+Alt+C` 全局快捷键截图，OCR 自动识别卡号、有效期、CVV
- **剪贴板识别** - `Ctrl+Alt+V` 从剪贴板识别图片或文本
- **手动输入** - 支持各种凌乱格式，多张卡片混排也能正确解析
- **精美卡片** - 根据卡组织（Visa / Mastercard / Amex / UnionPay 等）自动配色
- **二维码** - 每张卡片自动生成二维码，微信可扫
- **钉到桌面** - 卡片浮窗置顶，透明背景，可拖拽移动，右键菜单管理
- **双击放大** - 双击卡片查看大图，支持复制信息 / 复制图片 / 导出
- **搜索筛选** - 按卡号、品牌、有效期快速搜索
- **批量操作** - 全选 / 勾选，批量钉图、导出、删除
- **备份恢复** - 设置中一键导出 / 导入备份（卡片 + 设置）
- **系统托盘** - 关闭窗口自动最小化到托盘，后台常驻
- **开机自启** - 设置中一键开关
- **桌面通知** - 识别过程和结果通过右下角 Toast 通知提示
- **后台识别** - OCR 在后台线程执行，不卡 UI
- **快捷键可自定义** - 设置中修改截图和剪贴板的全局快捷键

## 快速开始

### 1. 初始化环境

双击 `setup_venv.bat`，自动创建虚拟环境并安装依赖。

或手动执行：

```bash
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
```

### 2. 启动应用

双击 `run.bat` 启动，或：

```bash
.venv\Scripts\activate.bat
python main.py
```

后台启动（直接进入托盘）：

```bash
python main.py --minimized
```

## 支持的输入格式

```
4111111111111111 12/25 123
4111 1111 1111 1111 | 12/25 | 123
卡号:4111111111111111 有效期:12/25 CVV:123
4111-1111-1111-1111, 1225, 123
5355...|03/29|456 <|> LIVE => bitcheck.cc5355...|07/30|789
```

## 打包为 EXE

使用 Nuitka 打包为单个独立 EXE（需全局安装 Nuitka）：

```bash
pip install nuitka
python build.py
```

输出 `CardSnap.exe`，包含所有依赖和资源，双击即可运行。

## 技术栈

- **UI**: PySide6 (Qt6)
- **OCR**: RapidOCR (onnxruntime)
- **二维码**: qrcode + Pillow
- **全局热键**: pynput
- **打包**: Nuitka
